# Lista Binary Search

1. Primeira Ocorrência: dado um vetor ordenado com repetidos, encontre o índice da primeira ocorrência de `x`

2. Última Ocorrência: dado um vetor ordenado com repetidos, encontre o índice da última ocorrência de `x`

3. Use pesquisa binária para encontrar quantas vezes um elemento `x` aparece no vetor

4. Menor elemento >= x: Encontre o menor valor do vetor que seja maior ou igual a `x`

5. Maior elemento <= x: Encontre o maior valor do vetor que seja menor ou igual a `x`

6. Um vetor foi ordenado e depois rotacionado:

```
[15, 18, 2, 3, 6, 12]
```

Encontre `x` em `O(log n)`

7. Encontre o menor elemento de um vetor ordenado e rotacionado
