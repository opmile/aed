# Matriz

---

Matriz é uma estrutura de dados. Conceitualmente, vetor tem uma dimensão e matriz tem mais de uma dimensão. Ainda sim, array (arranjo) é mencionado seja como vetor (array unidimensional) ou matriz (array multidimensional)

A matriz é arranjada de forma contígua na memória. Ao alocar espaço para uma matriz, precisamos de um espaço contínuo na memória.

---

Obs: passagem por valor (passa uma cópia) e por referência (passamos o ponteiro, o endereço da variável na memória) na função
